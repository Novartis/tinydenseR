# Color Palette

A color palette matrix used throughout the tinydenseR package for
consistent plotting.

## Usage

``` r
data(Color.Palette)
```

## Format

A matrix with 6 rows and 7 columns:

- Rows:

  Hue, Tint3, Tint2, Tint1, Shade1, Shade2

- Columns:

  Blue, Sienna, Apricot, Carmine, Gray, Light Gray, Dark Gray
